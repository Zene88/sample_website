Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most Important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a table
table table include a table
tr table row include a row of data
th table header include a table header
td table data include a table data cell
div div page division
header header container for introductory content
ol ordered list defines ordered list
ul unordered list defines unordered list
ls list defines a list element in an ordered/unordered list
blockquote blockquote marks a quote

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized texts <em>technical sophistication</em> technical sophistication
strong strong make bolt text <strong>technical sophistication</strong> technical sophistication
a anchor make link <a href="">technical sophistication</a> technical sophistication
img img insert image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span for styling <span>technical sophistication</span> technical sophistication

Document Tags

Tag Name Purpose Example
html html Defines the root of an HTML document <html>HTML Document</html>
head head Conains metadata of the document <head>Metadata</head>
title title Title of the webpage <title>title</title>
meta meta contains metadata <meta charset="UTF-8"/>
